# 🧪 Full Stack Alchemist

This repository was created as part of the admission phase for **Sentivue**, showcasing my skills in full-stack web development.

---

## 🛠️ Technologies Used

### 🧩 Frontend
- **Next.js** – React framework for server-side rendering and routing  
- **Tailwind CSS** – Utility-first CSS framework for styling

### 🔧 Backend
- **FastAPI** – High-performance Python web framework
- **PostgreSQL** – Relational database for storing transcribed text
- **AssemblyAI** – Public speech-to-text API used for audio transcription

---

## 🌐 Deployment

- **Frontend:** Hosted on [Vercel](https://vercel.com)
- **Backend:** Hosted on [Railway](https://railway.app)

🔗 **Live Site:** [https://full-stack-alchemist.vercel.app/](https://full-stack-alchemist.vercel.app/)

---

## 📄 Features

- Upload audio files (MP3, WAV, etc.)
- Transcribe audio using AssemblyAI API
- Store transcripts in a PostgreSQL database
- Clean, responsive UI built with Tailwind CSS
- export transcripts as SRT files
